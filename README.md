# AI Resume Screening System

An AI-powered Resume Screening Web Application built using Natural Language Processing (NLP) and Machine Learning techniques. The system analyzes resumes and compares them with job descriptions to generate an AI-based resume matching score.


## Features

- Upload Resume PDF
- Automatic Resume Text Extraction
- NLP-Based Text Preprocessing
- TF-IDF Vectorization
- Cosine Similarity Matching
- AI-Based Resume Match Percentage
- Interactive Streamlit Web Interface
- Real-Time Resume Analysis


## Technologies Used

Python, NLP, Scikit-learn, TF-IDF Vectorization, Cosine Similarity, NLTK, PyPDF2, Streamlit, Git, GitHub
